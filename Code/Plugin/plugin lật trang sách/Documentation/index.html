Installation

 

Flipper installs just like any other jQuery plugin:
 
  1. Copy the .css file to your css folder;
  2. Copy the .js file to your scripts folder;
  3. Include the files like this:
<link rel="stylesheet" href="css/flipper.css" type="text/css" /><script src="scripts/flipper.js"></script>
 
NOTE: Make sure that you include the script after you include jQuery!

How To Use

 

To start using flipper, you need to format your content like this:
 
<div id="your-flipper-id">    <div class="flipper-page">        The content for the first page goes here.    </div>    <div class="flipper-page">        The content for the second page goes here.    </div>    <div class="flipper-page">        The content for the third page goes here.    </div><!-- And so on --></div>
 
Finally, you need to call the script:
 
$('#your-flipper-id').flipper();
 
This is everything you need to do in order to get the plugin going. Additionally, you may need to specify some parameters:
 
$('#your-flipper-id').flipper({    "width" : 500,// The total width of the widget.    "height" : 250,// The total height of the widget.    "arrows" : true,// If you don't need the arrows, set this to false. Default: true    "pager" : true,// If you don't need the arrows, set this to false. Default: false    "imagesPath" : 'images/'// The path to your "images" folder, relative to the html                              // file, from which you include the script.                              // For example, in the demo it's relative to index.html.});
